Registration page

Typical Member Registration Flow

Here is how it will work.

  1. User visits our registration page
  2. User enters email address
  3. If it is new user
    1. Fills in registration form
    2. Submits the form
    3. If there is validation error
    4. Fix the validation error
    5. Review the fix
    6. Submit again
    7. If there is no error
    8. Submit the form
  4. If it is returning user
    1. Enter password
    2. If the entered password is wrong
    3. Ask password again
    4. If password is correct
    5. Good to go, proceed next step
    6. Confirm account info
  5. Confirm registration info

def block_code(code, language)
  case language
  when 'pseudo', 'pseudocode'
    wrap_svg Pseudocode.draw(code)
  when 'seq', 'sequence'
    wrap_svg Sequence.draw(code)
  else
    Pygments.highlight(code, lexer: language)
  end
end
